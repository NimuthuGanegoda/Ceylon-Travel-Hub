## Project: Sri Lanka RouteMaster
- **Tech Stack:** React, Tailwind, Vite (Adapted for Next.js environment).
- **Workflow:** Always create a new feature branch for tasks.
- **Coding Standard:** Use functional components and Lucide-React for icons.
- **Auto-Merge:** This repo uses a GitHub Action to auto-merge PRs that pass CI. Please ensure `npm test` or `npm run build` passes before pushing.
